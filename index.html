Ferramentas utilizadas

A ferramenta que iremos utilizar será o Visual Studio Code. O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft que permite escrever e editar códigos em várias linguagens de programação, como HTML, CSS, JavaScript, Python, entre outras. Ele é gratuito, de código aberto e pode ser usado em diferentes sistemas operacionais, como Windows, Mac e Linux.

O Visual Studio Code é um editor altamente personalizável e oferece diversos recursos avançados para ajudar os desenvolvedores a escreverem códigos com mais rapidez e eficiência, tais como sugestões de código, realce de sintaxe, indentação automática, depuração, controle de versão, integração com o terminal, além de uma ampla gama de extensões disponíveis que podem ser instaladas para expandir suas funcionalidades.

Devido à sua flexibilidade e ampla gama de recursos, o Visual Studio Code se tornou uma ferramenta popular entre desenvolvedores de todos os níveis de experiência e é amplamente utilizado na comunidade de desenvolvimento de software.

Inspertor de elementos

Para utilizar o inspetor de elementos, basta apertar a tecla F12 ou clicar com o botão direito do mouse na página da web que deseja inspecionar e selecionar a opção "Inspecionar" ou "Inspect". Isso abrirá o painel do inspetor de elementos, onde é possível visualizar e modificar o código HTML, CSS e JavaScript da página em tempo real.

O inspetor de elementos é uma ferramenta de desenvolvimento web integrada aos navegadores modernos que permite visualizar e modificar o código HTML, CSS e JavaScript de uma página em tempo real. Com essa ferramenta, é possível identificar problemas de layout, corrigir erros de código, testar e depurar código, entre outras funcionalidades. É uma ferramenta essencial para desenvolvedores web, designers e qualquer pessoa que queira entender como as páginas da web são construídas.

Estrutura basica do HTML

A estrutura básica do HTML é composta por dois elementos principais: o cabeçalho (head) e o corpo (body). O cabeçalho contém informações sobre a página, como o título e links para outros arquivos, enquanto o corpo contém o conteúdo principal da página, como texto, imagens e outros elementos. Essa estrutura é fundamental para garantir que o navegador interprete corretamente o código HTML e exiba o conteúdo da página da maneira desejada.

Falando sobre as tags

A tag strong é utilizada para indicar que o texto contido dentro dela é importante ou enfatizado, e deve ser exibido em negrito.

Tags Sugeridas

Tags que foram sugeridas para utilizar e aprender por conta propria pesquisando na internet.

A tag font é uma tag obsoleta em HTML que foi usada para definir o tamanho, a cor e a fonte do texto em uma página da web. Antigamente, era comum usar essa tag para estilizar o texto na web, mas hoje em dia é recomendado usar outras formas de estilização, como CSS.

O uso da tag font é bem simples. Você pode usar o atributo size para definir o tamanho da fonte, o atributo color para definir a cor da fonte e o atributo face para definir a fonte a ser usada. Uma das formas que poderia ser aplicado é por exemplo:

Este texto é grande, vermelho e em Arial.

A tag del é uma tag de marcação em HTML que é usada para indicar que um texto foi removido ou deletado. Geralmente, ela é usada para mostrar alterações feitas em um documento ou para indicar que um texto anteriormente mostrado já não é válido ou foi substituído. Um exemplo de uso da tag del seria o seguinte código HTML é: Eu gosto de futebol tênis.

A tag abbr é usada para definir uma abreviação ou sigla em um texto. Ela é usada para fornecer uma descrição completa ou uma explicação do significado da abreviação ou sigla. O atributo "title" é usado para fornecer a descrição completa. Exemplo:

O WHO é uma agência especializada das Nações Unidas que se preocupa com questões relacionadas à saúde em todo o mundo.

Neste exemplo, a abreviação "WHO" é definida usando a tag abbr e o atributo "title" fornece a descrição completa "World Health Organization". Quando o usuário passar o mouse sobre a abreviação "WHO", a descrição completa será exibida em uma caixa de ferramentas.

Atributos das tags

A tag ID é um atributo usado para identificar exclusivamente um elemento HTML em uma página. Ela é usada para permitir que os desenvolvedores se refiram a um elemento específico usando JavaScript ou CSS. O ID deve ser único para cada elemento em uma página HTML e é definido pelo atributo "id". É uma maneira útil de adicionar estilos e comportamentos específicos a um elemento HTML individualmente

A tag Style é usada para definir estilos em um documento HTML. Ela é usada para incluir as instruções CSS (Cascading Style Sheets) que definem como os elementos HTML devem ser exibidos na página. É possível incluir estilos internamente (no próprio documento HTML) ou externamente (em um arquivo separado). A tag Style inclui propriedades CSS como cores, fontes, tamanhos de fonte, margens, preenchimentos, entre outros.

A tag "type" é usada em várias tags HTML como script e input para especificar o tipo de conteúdo que está sendo definido ou referenciado. Por exemplo, no caso de script, o atributo "type" especifica o tipo de conteúdo do script, como JavaScript ou CSS. No caso de input, o atributo "type" especifica o tipo de entrada que é permitido no campo de entrada, como texto, senha ou botão.

O atributo "src" é usado em várias tags HTML, como img e script, para especificar o caminho do arquivo que contém o conteúdo a ser exibido ou executado. Por exemplo, no caso de img, o atributo "src" especifica o caminho da imagem a ser exibida na página, enquanto no caso de script, o atributo "src" especifica o caminho do arquivo JavaScript a ser executado.

O atributo "width" é utilizado em diversas tags HTML, como img e table, para definir a largura do elemento em pixels ou em porcentagem da largura da janela de exibição. Em img, o atributo "width" determina a largura da imagem que será exibida na página. Já em table, o atributo "width" define a largura da tabela em pixels ou porcentagem da largura da janela de exibição.

Textos

As tags h1 a h6 são utilizadas para criar títulos e subtítulos em uma página HTML, com a tag h1 sendo a mais importante (maior) e a h6 a menos importante (menor). A utilização dessas tags permite estruturar e organizar o conteúdo da página, além de fornecer informações sobre a hierarquia de informações do documento. Cada tag é estilizada com um tamanho de fonte diferente, sendo que a h1 tem o maior tamanho e a h6 o menor. Resumidamente, as tags h1 a h6 são usadas para criar títulos e subtítulos em uma página web, com a h1 sendo a mais importante (maior) e a h6 a menos importante (menor).

A tag "p" é usada em HTML para definir um parágrafo de texto. Ele é usado para separar o texto em seções ou parágrafos distintos para melhorar a legibilidade e organização do conteúdo em uma página da web. Ao usar a tag "p", o navegador adiciona automaticamente um espaço em branco antes e depois do parágrafo, criando uma aparência visualmente agradável na página. É uma das tags mais comuns usadas em HTML para formatar o conteúdo de uma página da web.

A tag "blockquote" é usada em HTML para indicar uma citação ou trecho de texto que foi retirado de outra fonte. Ele é usado para destacar o texto e indicar que ele não é original para a página da web. O navegador irá adicionar automaticamente um recuo para o texto dentro da tag, criando uma aparência visualmente distintiva. É comumente usado em artigos e blogs para citar outras fontes de informação.

A tag "mark" é usada em HTML para destacar ou realçar partes do texto em uma página da web. É semelhante a um marca-texto, onde é possível escolher uma cor para destacar o texto. A tag é útil para destacar informações importantes em um texto ou para destacar trechos de citações de outras fontes. Quando a tag é usada, o texto destacado fica com um fundo amarelo por padrão, mas a cor de fundo pode ser modificada por meio de CSS.

A tag "sup" é usada em HTML para criar um texto sobrescrito, o que significa que o texto aparecerá acima da linha de base normal e ligeiramente menor em tamanho. Ele pode ser usado para fins de notação matemática, como exponenciação, ou para criar referências a notas de rodapé ou abreviações. Ao usar a tag "sup", o navegador adiciona automaticamente o estilo apropriado para o texto sobrescrito. Exemplo: O valor de 102 é 100.

A tag "sub" em HTML é usada para indicar texto que deve ser exibido como uma subscrito, ou seja, um pouco abaixo da linha de base normal do texto. Ela é frequentemente usada para exibir fórmulas químicas ou matemáticas, notas de rodapé e outras informações que precisam ser apresentadas em uma posição ligeiramente inferior à linha de texto principal. A tag "sub" deve ser usada em conjunto com a tag "sup" para indicar texto que deve ser exibido como sobrescrito. Exemplo:

Listas ordenadas e não ordenadas

A tag "ol" é usada em HTML para criar uma lista ordenada, onde cada item é identificado por um número ou letra em sequência. É uma maneira comum de apresentar informações em uma lista estruturada e organizada. A tag "ol" deve ser usada em conjunto com a tag "li", que define cada item da lista. Alguns dos atributos mais comuns que podem ser usados com a tag "ol" incluem "start", que especifica o número inicial da lista, e "type", que permite que a lista seja formatada com diferentes tipos de marcadores, como números, letras ou romanos.

A tag "ul" é usada em HTML para criar uma lista não ordenada. Ela é usada para agrupar uma lista de itens que não precisam ser ordenados em uma sequência específica, como uma lista de marcadores ou itens em tópicos diferentes. Cada item na lista é criado usando a tag "li", que significa "item de lista". Ao usar a tag "ul", o navegador exibe automaticamente marcadores para cada item na lista, mas é possível personalizar o estilo desses marcadores usando CSS.

Links

tag "a" é usada para criar um link em HTML. O atributo "href" é usado para especificar o URL da página para a qual o link aponta. Ao clicar no link, o usuário será redirecionado para a página especificada no atributo "href". Além disso, a tag "a" pode ser usada para criar links internos em uma página, permitindo que os usuários naveguem facilmente entre diferentes seções da página. É uma das tags mais comuns usadas em HTML para conectar páginas da web.

Criar uma pagina web utilizando as tags aprendidas

A tag "hr" em HTML é usada para criar uma linha horizontal na página da web, que é frequentemente usada como uma separação visual entre seções de conteúdo. É uma tag de auto-fechamento e não possui conteúdo entre as tags de abertura e fechamento. O atributo "size" pode ser usado para especificar o tamanho da linha horizontal e o atributo "width" pode ser usado para especificar a largura da linha em pixels ou porcentagem da largura da janela de exibição.

A tag "small" é usada em HTML para especificar que o texto deve ser exibido em um tamanho menor do que o texto circundante. Geralmente, é usado para incluir textos menores, como avisos de direitos autorais, notas de rodapé ou outros textos secundários que não precisam ser tão proeminentes quanto o texto principal na página da web. Quando usado, o navegador diminui o tamanho da fonte para o valor especificado na tag "small". Por exemplo; Esta frase esta com a tag "small" aplicada."
